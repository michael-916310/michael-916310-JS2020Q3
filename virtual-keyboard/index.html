Virtual Keyboard w/ HTML, CSS & JS

Поле ввода:
  • Для подсветки клавиш на виртуальной клавиатуре (и прогрывания звука) при нажатии на компьютерной клавиатуре:
    • Раскладки (ENG\RUS + SHIFT) виртуальной и компьютерной клавиатуры должны совпадать. Иначе на вирутальной клавиатуре может не найтись символ нажатый на компьютерной.
    • Фокус (курсор) должен быть на поле ввода. Если фокус убрать с поля ввода убрать с поля ввода, нет смысла отслеживать нажатие на реальной клавиатуре, т.к. такие нажатия не понятно к чему относятся.
  • Responsive
  • Vanilla JS (no libraries required!)
